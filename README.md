# 智课笔记

> 基于大语言模型的智能学习助手，让每一堂课都自动生成高质量笔记

## 核心亮点

### ☁️ 云存储 - 数据永不丢失
- **自动同步**：数据实时云端备份
- **多端同步**：手机、平板、电脑数据共享
- **离线可用**：无网络时自动切换本地模式
- **安全可靠**：私有数据，他人无法访问

### 🤖 AI智能 - 学习效率翻倍
- **录音转写**：课堂录音自动转文字
- **AI答疑**：基于笔记内容智能回答
- **RAG知识库**：官方资料+用户笔记双重知识源
- **智能复习**：自动生成试卷、背诵卡片

### 📚 学习管理 - 知识体系化
- **课程管理**：自由创建和管理课程
- **笔记系统**：多种创建方式，内容持续累积
- **错题本**：自动收集错题，针对性练习
- **急救模式**：期末知识点快速复习

## 快速开始

### 1. 创建课程
```
首页 → 点击"+"按钮 → 选择"创建课程" → 输入课程名称
```

### 2. 创建笔记（四种方式）
- **手动创建**：首页"+" → "新建笔记"
- **录音转写**：底部Tab"录音" → 选择课程 → 开始录音
- **拍图识别**：首页"拍图识别" → 拍照/相册
- **课程内创建**：点击课程 → "+ 新建"

### 3. 使用AI功能
- **AI答疑**：笔记详情 → "AI答疑"
- **生成试卷**：笔记详情 → "生成试卷"
- **背诵卡片**：笔记详情 → "背诵卡片"

### 4. 复习训练
```
底部Tab"复习" → 选择模式 → 选择课程 → 开始学习
```

### 5. 云存储管理
```
"我的" → "云存储管理" → 查看状态、同步数据
```

## 功能详解

### 📝 笔记系统
- **手动创建**：适合整理笔记
- **录音转写**：适合课堂录音，可选择追加到已有笔记
- **拍图识别**：适合拍板书、教材
- **课程内创建**：在课程中快速创建

### 🎤 录音转写
- 选择课程后可选择笔记
- 选择笔记：转写结果追加到该笔记
- 不选择：创建新笔记

### 🤖 AI答疑
基于RAG技术，组合用户笔记和官方知识库进行智能回答

### 📚 复习模块
1. **复习卷**：选择课程 → 配置题型 → 答题
2. **背诵卡片**：选择课程 → 自动生成Anki式卡片
3. **急救模式**：选择课程 → 生成知识点总结
4. **错题本**：查看所有错题，支持编辑删除

### ☁️ 云存储
- 查看云端/本地数据统计
- 手动同步数据
- 从云端恢复数据
- 导出/导入数据（JSON）

## 配置说明

### API配置（可选）
如需使用AI功能，需配置Coze API：
1. 访问 https://www.coze.cn
2. 获取Token和Bot ID
3. "我的" → "API配置" → 填写并保存

### 云开发配置
小程序已内置云开发配置，自动启用云存储。
