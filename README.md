# 智课笔记 (AI Smart Class Notes)

> 基于大语言模型的智能笔记与答疑平台

## 📱 项目简介

智课笔记是一款专为高校学生设计的智能笔记工具，利用大语言模型技术实现课堂录音转写、AI答疑、复习卷生成等功能，帮助学生提高学习效率。

## ✨ 核心功能

### 1. 课堂录音转写 🎤
- 实时录音并自动转写为文字
- 基于 Coze 大模型生成高质量笔记
- 自动提取关键知识点和总结

### 2. 拍图识别 📷
- 拍照识别黑板、PPT内容
- 支持手写笔记识别
- 一键保存为笔记

### 3. AI 答疑 💬
- 基于笔记内容的智能问答
- 支持上下文理解
- 历史问答记录保存

### 4. 复习卷生成 📝
- 根据笔记自动生成练习题
- 支持多种题型（选择/填空/问答）
- 难度可调

### 5. 错题本 ❌
- 自动收录复习卷中的错题
- 分类管理，便于复习
- 支持标记掌握状态

### 6. 思维导图 🗺️
- 自动生成知识点思维导图
- 可视化知识结构
- 支持导出

### 7. 急救模式 🚑
- 将笔记压缩为2页纸精华
- 考前快速复习利器

### 8. 背诵卡片 📚
- Anki式记忆卡片
- 支持翻转查看答案
- 标记掌握程度
- 智能复习安排

## 🚀 快速开始

### 环境要求
- 微信开发者工具
- 微信小程序基础库 2.19.0+

### 安装步骤

1. 克隆项目
```bash
git clone <项目地址>
cd 智课笔记-dev/miniprogram
```

2. 使用微信开发者工具打开 `miniprogram` 目录

3. 在 `app.js` 中配置 Coze API（可选，未配置时使用模拟数据）

4. 编译运行

### Coze API 配置

1. 访问 [Coze 平台](https://www.coze.cn)
2. 创建开发者令牌（Token）
3. 创建以下 Bot：
   - 笔记总结 Bot
   - 答疑助手 Bot
   - 试卷生成 Bot
   - 卡片生成 Bot
4. 在小程序"个人中心"->"API配置"中填入 Token 和 Bot ID
