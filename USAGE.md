# 智课笔记 - 使用指南

## 快速开始

### 第一次使用

1. **打开小程序**
   - 显示欢迎引导
   - 介绍核心功能

2. **开始使用**
   - 首页为空白画布
   - 点击右下角"+"按钮开始创建

### 创建课程

```
首页 → "+"按钮 → "创建课程" → 输入课程名称 → 保存
```

## 笔记功能

### 创建笔记（四种方式）

#### 1. 手动创建
```
首页 → "+" → "新建笔记" → 选择课程 → 输入内容 → 保存
```

#### 2. 录音转写
```
底部Tab"录音" → 选择课程 → (可选)选择笔记 → 开始录音 → 结束录音 → 转写
```

**提示**：
- 选择已有笔记：转写内容追加到该笔记
- 不选择笔记：创建新笔记

#### 3. 拍图识别
```
首页 → "拍图识别" → 拍照/相册 → 识别 → 保存为笔记
```

#### 4. 课程内创建
```
点击课程卡片 → "+ 新建" → 输入内容 → 保存
```

### 笔记内容

笔记会自动累积以下内容：
- 原始内容（手写或转写）
- AI问答记录（自动追加）
- 录音转写内容（自动追加）
- 图片识别内容（自动追加）

## AI功能

### AI答疑

```
笔记详情 → "AI答疑" → 输入问题 → 查看回答
```

**特点**：
- 基于笔记内容回答
- 结合官方知识库
- 自动保存问答记录

### 生成试卷

```
笔记详情 → "生成试卷" → 配置题型/题量/难度 → 生成
```

**答题流程**：
1. 选择题型（选择/填空/简答）
2. 选择题量（5-20题）
3. 选择难度（简单/中等/困难）
4. 开始答题
5. 提交查看成绩
6. 错题自动加入错题本

### 背诵卡片

```
笔记详情 → "背诵卡片" → 自动生成 → 开始学习
```

**使用方式**：
- 点击卡片翻转查看答案
- 标记掌握程度
- 智能复习安排

## 复习模块

### 进入复习

```
底部Tab"复习" → 选择模式
```

### 四种复习模式

#### 1. 复习卷
```
选择"复习卷" → 选择课程 → 配置 → 生成试卷 → 答题
```

#### 2. 背诵卡片
```
选择"背诵卡片" → 选择课程 → 自动生成卡片 → 开始学习
```

#### 3. 急救模式
```
选择"急救模式" → 选择课程 → 生成知识点总结
```

**内容包括**：
- 核心概念
- 重点公式
- 常见题型
- 易错点

#### 4. 错题本
```
选择"错题本" → 查看所有错题 → 编辑/删除
```

## 云存储管理

### 查看云存储状态

```
"我的" → "云存储管理"
```

**显示内容**：
- 云连接状态
- 云端数据统计
- 本地数据统计
- 存储空间使用
- 待同步数据数量

### 数据同步

#### 自动同步
- 创建/修改数据时自动同步
- 网络恢复自动提示同步

#### 手动同步
```
云存储管理 → "上传所有数据" / "同步待上传数据"
```

### 数据恢复

```
云存储管理 → "从云端恢复" → 确认 → 下载
```

### 数据备份

#### 导出数据
```
云存储管理 → "导出数据" → 生成JSON文件
```

#### 导入数据
```
云存储管理 → "导入数据" → 选择JSON文件 → 导入
```

## 常见问题

### Q: 如何创建第一门课程？
A: 首页点击"+"按钮，选择"创建课程"，输入课程名称。

### Q: 录音转写会创建新笔记吗？
A: 可以选择。选择已有笔记会追加内容，不选择则创建新笔记。

### Q: 离线能用吗？
A: 可以。所有功能离线可用，数据会自动保存到本地，网络恢复后同步到云端。

### Q: 数据会丢失吗？
A: 不会。数据同时保存在本地和云端，建议定期查看云存储状态。

### Q: 如何查看AI问答记录？
A: 在笔记详情页，AI问答记录会自动追加到笔记内容中。

### Q: 错题本如何使用？
A: 复习卷答题错误会自动加入错题本，可以查看、编辑、删除错题。

### Q: 如何配置AI功能？
A: "我的" → "API配置" → 填写Coze Token和Bot ID。

### Q: 云存储收费吗？
A: 云开发提供免费额度，个人使用完全够用。

## 使用建议

### 数据安全
1. 定期查看云存储状态
2. 重要数据及时同步
3. 定期导出数据备份

### 学习效率
1. 课后及时创建笔记
2. 善用AI答疑功能
3. 定期使用复习模块
4. 重视错题本复习

### 最佳实践
1. 为每门课创建一个课程
2. 一门课可以有多个笔记
3. 录音时选择已有笔记追加内容
4. 定期整理和复习
