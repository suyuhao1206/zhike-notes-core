# 图标资源说明

此文件夹需要放置 tabBar 图标资源，需要以下文件：

## 必需文件

| 文件名 | 用途 |
|--------|------|
| home.png | 首页未选中图标 |
| home-active.png | 首页选中图标 |
| mic.png | 录音未选中图标 |
| mic-active.png | 录音选中图标 |
| review.png | 复习未选中图标 |
| review-active.png | 复习选中图标 |
| profile.png | 我的未选中图标 |
| profile-active.png | 我的选中图标 |

## 图标规格

- 大小：81px × 81px
- 格式：PNG
- 透明背景
- 颜色：未选中 #999999，选中 #1890FF

## 获取图标

1. 从 [阿里巴巴矢量图标库](https://www.iconfont.cn/) 下载
2. 或使用微信官方示例图标
3. 在线生成工具制作
